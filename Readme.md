\# Python Programming Tasks



\## Description

This repository contains four Python projects completed as part of internship training. The projects demonstrate fundamental programming concepts including problem-solving, file handling, regular expressions, user interaction, and basic application development.



\## Projects



\### Task 1: Python Hangman Game



A word-guessing game where the user attempts to identify a hidden word within a limited number of attempts.



\### Task 2: Stock Portfolio Tracker



A program that calculates the total value of a stock portfolio using predefined stock prices and user-provided quantities.



\### Task 3: Email Extractor



A utility that extracts email addresses from a text file and stores the extracted results in a separate file.



\### Task 4: Basic Chatbot



A rule-based chatbot that responds to predefined user inputs using conditional statements and loops.



\## Technologies Used



\* Python 3

\* File Handling

\* Regular Expressions

\* Functions

\* Loops

\* Conditional Statements
